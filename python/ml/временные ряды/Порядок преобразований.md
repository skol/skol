# Порядок преобразований — имеет значение.
Сначала **стабилизируем дисперсию**, потом **убираем тренд**.
## Почему такой порядок?
**Шаг 1: Стабилизация дисперсии**
- Если дисперсия растет вместе с уровнем ряда (например, ряд идет вверх и "размах" колебаний увеличивается), это называется **гетероскедастичность**   
- Такие ряды часто встречаются в экономике, финансах, демографии    
- **Преобразования:** логарифмирование (`log`), Box-Cox, иногда квадратный корень

**Зачем сначала:** если сначала взять разность (убрать тренд), дисперсия может остаться нестабильной, и это можно не заметить на графике, а модели потом "поплывут".

**Шаг 2: Удаление тренда**
- После стабилизации дисперсии берем разности (обычно первую)    
- Если ряд был с сезонностью — возможно, потребуется сезонная разность
## Пример
```python
# Правильно
df['log'] = np.log(df['value'])
df['diff'] = df['log'].diff()

# Неоптимально (если дисперсия росла)
df['diff'] = df['value'].diff()
# дисперсия может остаться нестабильной, особенно если ряд сильно вырос
```

## Когда можно менять порядок?
- Если дисперсия **визуально стабильна** (ряд не расширяется со временем) — можно и сразу разность    
- Если ряд **уже стационарен по дисперсии**, но есть тренд — достаточно разности    
- Если есть **сезонность** — часто сначала берут сезонную разность, потом обычную
